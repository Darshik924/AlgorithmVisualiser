# 📊 Algorithm Visualizer

This project was developed as part of the **Winter 2025 assignment for CSESA**.

It is an **Algorithm Visualizer** designed to help users better understand how sorting algorithms work through **interactive animations and visual feedback**.

---

## 🚀 Overview

Understanding sorting algorithms can be challenging when approached only through code.  
This project simplifies the learning process by providing **step-by-step visual representations** of how algorithms operate internally.

Through dynamic animations and color-coded transitions, users can clearly observe how elements are compared, swapped, and sorted.

---

## ✨ Features

- 📌 **Sorting Algorithms Implemented**
  - Bubble Sort
  - Selection Sort
  - Quick Sort

- 📥 **Custom Input Support**
  - Users can define the size of the array
  - Input custom values in any order

- 🎬 **Real-Time Visualization**
  - Elements are displayed as bars/blocks
  - Animations show comparisons and swaps
  - Color changes highlight algorithm steps

- 🎯 **Interactive Learning**
  - Helps in building intuition for sorting logic
  - Makes it easier to recall and implement algorithms later

---

## 🛠 How It Works

1. Enter the **size of the array**
2. Provide the **array elements** (in any order)
3. Select the **sorting algorithm**
4. Click **"Go"** to start visualization
5. Watch the elements animate and get sorted step-by-step
